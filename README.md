## 📌 About KeenKeeper

> **KeenKeeper** is a modern, intuitive expense tracking dashboard built to give you crystal-clear visibility into your spending habits. Log transactions, analyze trends with beautiful charts, and take control of your finances — all in one clean interface.

Whether you're budgeting for the month or auditing past spending, KeenKeeper makes financial awareness effortless and visually engaging.

---

## ✨ Key Features

### 📊 Interactive Spending Analytics
Visualize your expenses through dynamic charts powered by **Recharts**. Track spending by category, compare monthly trends, and get instant insight into where your money is going — with responsive, animated charts that work across all screen sizes.

### ⚡ Instant Transaction Management
Add, edit, and delete expense records in real-time with live UI feedback via **React Toastify** notifications. Every action is smooth, snappy, and confirmed — no page reloads, no confusion.

### 💾 Persistent Local Storage
Your data stays with you. KeenKeeper is being upgraded with **vanilla JS LocalStorage** integration to persist all transactions across sessions — no account or backend required.

## 🚀 Getting Started

### Prerequisites
- Node.js `v18+`
- npm / yarn / pnpm / bun

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/wazed-md-abdul/KeenKeeper.git

# 2. Navigate into the project
cd KeenKeeper

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see KeenKeeper in action.
